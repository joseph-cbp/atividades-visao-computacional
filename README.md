
# Trilha de Visão Computacional (Computer Vision Track)

Este repositório contém a exploração das atividades complementáres dos módulos da Trilha de Visão Computacional da NexVisual em parceria com o Instituto Eldorado.
As atividades complementares estão localizadas na parte de baixo de cada arquivo .ipynb.

---

## Tecnologias e Ferramentas

* **Linguagem:** Python
* **Ambiente:** Jupyter Notebook
* **Processamento Computacional & Vetorização:** NumPy
* **Visualização & Análise Exploratória:** Matplotlib (plt)
* **Processamento de Imagem Tradicional:** OpenCV
* **Modelos & Frameworks:** YOLO, GANs, CLIP, Segmentação Semântica (PyTorch/TensorFlow)

---

## Como Executar os Notebooks

### Pré-requisitos
- Python3
- VSCode

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/joseph-cbp/atividades-visao-computacional.git
   cd atividades-visao-computacional
   ```
2. **Criar um ambiente virtual (opcional)**
   ```bash
   python -m venv venv
   # No macOS/Linux:
   source venv/bin/activate
   # No Windows:
   venv\Scripts\activate
   ```
3. **Executando no VSCode**
  - Abra a pasta do projeto no VSCode.
  - Abra o arquivo .ipynb (Notebook).
  - No canto superior direito do VSCode, clique em Select Kernel (Selecionar Kernel) e escolha o ambiente virtual venv criado no passo anterior.
  - Pronto! Agora basta executar as primeiras células de cada notebook — nelas já estão inclusos os comandos de pip install e os imports necessários para rodar a atividade automaticamente.
